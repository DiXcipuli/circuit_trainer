# Circuit Trainer

Made to automatize a training session: Plays a song, displays and times each exercice, and beeps when each is over. 

## Components
- Arduino Uno
- Sd card reader
- LCD display (2*16, raw connections, no I2C)
- Audio amplifier
- Piezo Buzzer
